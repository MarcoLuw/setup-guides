# ClickHouse Cluster with Docker
This guide provides instructions to set up a ClickHouse cluster using Docker containers for testing and development purposes. <br />

This setup is provided by ClickHouse Official Docs, for more details, please refer to: https://clickhouse.com/docs/architecture/replication

## Prerequisites
- Docker engine 29.1.3
- Docker Compose plugin v5.0.0
- Docker Buildx plugin v0.30.1

## Steps to Set Up ClickHouse Cluster
```bash
docker compose up -d
```
